# Flow Universe — Руководство пользователя

> **Аудитория:** PM, аналитики, разработчики, CIO
> **Версия:** 0.1 (MVP)
> **Обновлено:** 2026-03-25

---

## Что такое Flow Universe?

**Flow Universe** — система управления проектами и задачами для команд от 50 до 5000 человек. Разработана как замена Jira для российского финансового сектора.

Вы можете:
- Создавать проекты и задачи с иерархией (Эпик → История → Задача → Подзадача, Баг)
- Вести Kanban-доски и спринты
- Учитывать время с помощью таймера или вручную
- Смотреть отчёты по статусам и исполнителям
- Подключить GitLab для автоматического обновления статусов задач

## Вход в систему

1. Откройте ссылку на Flow Universe в браузере (Chrome 139+, Yandex Browser 25+, Edge 139+, Safari 18+)
2. Введите **email** и **пароль**
3. Нажмите **Войти**

Если вы забыли пароль — обратитесь к администратору системы.

## Навигация

После входа вы попадаете на **Дашборд**. В левой панели — главное меню:

| Пункт меню | Что открывает |
|-----------|--------------|
| Дашборд | Сводка активности и ключевые метрики |
| Проекты | Список всех проектов |
| Спринты | Все спринты по всем проектам |
| Моё время | Ваши записи о времени |
| Команды | Список команд |
| Администрирование | Панель администратора (только ADMIN+) |

## Ролевая модель

В Flow Universe пять ролей с разными правами доступа:

| Роль | Кто обычно | Что может |
|------|-----------|----------|
| **VIEWER** | CIO, руководитель | Просмотр проектов, задач, отчётов — только чтение |
| **USER** | Разработчик | Свои задачи + учёт времени |
| **MANAGER** | PM, тимлид | Всё + управление проектами и спринтами |
| **ADMIN** | Системный администратор | Всё + управление пользователями |
| **SUPER_ADMIN** | Технический администратор | Полный доступ |

Подробнее по каждой роли — в разделах:
- [roles/viewer.md](roles/viewer.md) — для CIO и руководителей
- [roles/user.md](roles/user.md) — для разработчиков и аналитиков
- [roles/manager.md](roles/manager.md) — для PM и тимлидов
- [roles/admin.md](roles/admin.md) — для администраторов

## Быстрый старт (5 минут)

### Разработчику

1. Войдите с ролью USER (dev@tasktime.ru)
2. Откройте **Проекты** → выберите проект
3. Найдите свою задачу в списке (или через фильтр «Исполнитель: Я»)
4. Нажмите **▶ Старт** рядом с задачей — запустится таймер
5. Когда закончите — нажмите **■ Стоп** — время сохранится
6. Смените статус задачи на «На ревью» через карточку задачи

### Менеджеру

1. Войдите с ролью MANAGER
2. Откройте **Проекты** → создайте проект (кнопка «+ Проект»)
3. Создайте задачи (кнопка «+ Задача» в списке проекта)
4. Откройте **Доску** проекта — увидите задачи по колонкам
5. Создайте спринт: вкладка **Спринты** → «+ Спринт» → назначьте задачи → «Старт»

### CIO / Viewer

1. Войдите с ролью VIEWER
2. Откройте **Администрирование** → вкладка **Отчёты**
3. Выберите проект → увидите задачи по статусам и по исполнителям
