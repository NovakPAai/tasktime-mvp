# Учёт времени — Руководство пользователя

> Обновлено: 2026-03-25

---

## Таймер

### Запуск таймера

**Способ 1: На карточке задачи**
1. Откройте задачу
2. Нажмите кнопку **▶ Старт** (в разделе «Время»)

**Способ 2: В списке задач**
- Нажмите кнопку **▶** рядом с задачей

После запуска:
- В шапке приложения появится счётчик (00:00:00)
- Кнопка превращается в **■ Стоп**

**Ограничение:** Одновременно можно вести только **один таймер**. При попытке запустить второй — система предложит остановить текущий.

### Остановка таймера

Нажмите **■ Стоп** (на карточке задачи или в шапке приложения).

Система автоматически создаст запись времени с:
- Датой
- Продолжительностью (в часах с точностью до 0,01)
- Ссылкой на задачу

## Ручной ввод времени

Если вы забыли запустить таймер или хотите добавить время за прошлый день:

1. Откройте задачу → раздел **Время**
2. Нажмите **+ Добавить вручную**
3. Заполните форму:

| Поле | Обязательно | Описание |
|------|-------------|---------|
| **Часы** | ✓ | Например: 2.5 (два с половиной часа) |
| **Дата** | — | По умолчанию: сегодня |
| **Примечание** | — | Что делали |

4. Нажмите **Сохранить**

---

## Мои записи времени

Страница **«Моё время»** (меню → **Время**):

- Показывает все ваши записи
- **Фильтры:** по дате (от/до), по задаче
- **Итог по дням** — сколько часов в каждый день
- **Итог по задачам** — сколько потрачено на каждую задачу
- Кнопка **Удалить** — удалить запись (только свои)

---

## Время на карточке задачи

В разделе «Время» карточки задачи отображается:

| Поле | Описание |
|------|---------|
| **Оценка** | Плановые часы (указывается при создании/редактировании) |
| **Затрачено** | Сумма всех записей времени по этой задаче |
| **Записи** | Список всех time-логов (кто, когда, сколько) |

Администратор и менеджер видят время всех пользователей по задаче.

---

## Удаление записи времени

1. Откройте задачу → раздел **Время**
2. Найдите запись → нажмите **✕**
3. Подтвердите удаление

**Кто может удалять:** автор записи или ADMIN.

---

## AI-логи времени

Если задача выполнялась AI-агентом, записи времени создаются автоматически с пометкой **AGENT** (источник). Стоимость AI-сессии также отображается рядом с записью.
