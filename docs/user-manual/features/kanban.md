# Kanban-доска — Руководство пользователя

> Обновлено: 2026-03-25

---

## Открытие доски

1. Откройте **Проекты** → выберите проект
2. Нажмите кнопку **Доска** (или перейдите по ссылке `/projects/:id/board`)

---

## Структура доски

Доска состоит из **5 колонок**, соответствующих статусам задач:

```
┌──────────┬────────────┬──────────┬──────────┬────────────┐
│  OPEN    │IN_PROGRESS │  REVIEW  │   DONE   │ CANCELLED  │
│          │            │          │          │            │
│ [TTMP-1] │ [TTMP-3]  │ [TTMP-5] │ [TTMP-7] │            │
│ [TTMP-2] │ [TTMP-4]  │          │ [TTMP-8] │            │
└──────────┴────────────┴──────────┴──────────┴────────────┘
```

Каждая карточка показывает:
- Ключ задачи (TTMP-42)
- Название
- Тип (иконка)
- Приоритет (цветная метка)
- Исполнитель (аватар)
- Оценку и затраченное время (если заполнены)

---

## Перемещение задач (Drag & Drop)

1. Зажмите карточку задачи левой кнопкой мыши
2. Перетащите в нужную колонку
3. Отпустите — статус сохраняется автоматически

Внутри колонки можно менять порядок задач перетаскиванием.

**Ограничения:**
- Нельзя перетащить задачу в колонку, если у вас нет права на изменение статуса
- USER может менять статус только своих задач

---

## Быстрые действия на карточке

Наведите мышь на карточку — появятся кнопки:
- **Открыть** — перейти на полную карточку задачи
- **Статус** — быстрое изменение через выпадающий список
- **Назначить** — изменить исполнителя (MANAGER, ADMIN)

---

## Фильтрация на доске

В шапке доски есть фильтры:
- **Исполнитель** — показать задачи только конкретного человека
- **Спринт** — показать задачи только выбранного спринта
- **Приоритет** — фильтр по приоритету

---

## Советы

- Для большого количества задач используйте фильтр «Спринт: текущий» — доска будет компактнее
- Колонка DONE показывает только задачи из текущего (активного) спринта
- Задачи без спринта также отображаются на доске (из бэклога)
