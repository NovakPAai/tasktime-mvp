# Спринты — Руководство пользователя

> Обновлено: 2026-03-25

---

## Что такое спринт?

Спринт — ограниченный по времени период работы (обычно 1-2 недели), в течение которого команда выполняет набор задач из бэклога.

В Flow Universe каждый проект может иметь только **один активный спринт** одновременно.

---

## Просмотр спринтов

**Спринты проекта:**
1. Откройте **Проекты** → выберите проект → вкладка **Спринты**

**Все спринты (глобально):**
- Меню слева → **Спринты** → откроется `/sprints`

---

## Состояния спринта

| Состояние | Описание |
|-----------|---------|
| **PLANNED** | Создан, задачи назначаются |
| **ACTIVE** | В процессе выполнения. Один на проект |
| **CLOSED** | Завершён |

---

## Создание спринта

Требуется роль **MANAGER** или **ADMIN**.

1. Откройте вкладку **Спринты** в проекте
2. Нажмите **+ Спринт**
3. Заполните:

| Поле | Обязательно | Описание |
|------|-------------|---------|
| **Название** | ✓ | Например: «Sprint 4» |
| **Цель** | — | Что хотим достичь |
| **Дата начала** | — | Плановая дата старта |
| **Дата окончания** | — | Плановая дата завершения |

4. Нажмите **Создать**

---

## Добавление задач в спринт

**Способ 1: Через планирование спринта**
1. Откройте спринт → нажмите **Планировать**
2. В `SprintPlanningDrawer` выберите задачи из бэклога
3. Нажмите **Добавить в спринт**

**Способ 2: На карточке задачи**
1. Откройте задачу
2. Поле **Спринт** → выберите нужный спринт из списка

---

## Запуск спринта

Требуется роль **MANAGER** или **ADMIN**.

1. Откройте спринт → нажмите **Старт**
2. Подтвердите — спринт переходит в состояние **ACTIVE**

**Важно:** Если у проекта уже есть активный спринт — новый запустить нельзя. Сначала закройте текущий.

---

## Закрытие спринта

1. Откройте активный спринт → нажмите **Закрыть**
2. Система покажет незавершённые задачи
3. Выберите: перенести в следующий спринт или оставить в бэклоге
4. Нажмите **Закрыть спринт**

---

## Просмотр содержимого спринта

Нажмите на название спринта — откроется `SprintIssuesDrawer` со списком задач:
- Название задачи
- Статус
- Исполнитель
- Затраченное / плановое время
- Прогресс-бар спринта

---

## Метрики спринта

На странице спринта отображаются:
- **Всего задач** в спринте
- **Готово** (статус DONE)
- **В работе** (статус IN_PROGRESS)
- **Оставшихся** (статус OPEN)
- **Прогресс** (% выполненных задач)
