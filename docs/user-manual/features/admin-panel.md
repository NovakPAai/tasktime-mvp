## Вкладка: Пользователи (только ADMIN)

**Список пользователей** с колонками:
- Имя и email
- Роль
- Статус (активен / заблокирован)
- Дата регистрации
- Последняя активность
- Количество действий в аудит-логе

### Создать пользователя
1. Нажмите **+ Пользователь**
2. Заполните email, имя, пароль, роль
3. Нажмите **Создать**

### Изменить роль
1. Найдите пользователя в таблице
2. Нажмите **Изменить** (иконка карандаша)
3. Выберите новую роль

**Ограничения:**
- Нельзя изменить собственный аккаунт
- ADMIN не может назначить роль SUPER_ADMIN
- Только SUPER_ADMIN может назначать роль ADMIN

### Заблокировать / разблокировать пользователя
- Нажмите на переключатель в колонке «Статус»
- Заблокированный пользователь не может войти в систему (JWT не принимается)

## Раздел: Система (только SUPER_ADMIN)

> Путь: Меню → Администрирование → Система (`/admin/system`)

Доступен исключительно пользователям с ролью **Супер Администратор**.

### Сессия пользователя

Настройка скользящей сессии — автоматический выход при бездействии.

| Поле | Описание |
|------|----------|
| Время бездействия до выхода | Количество минут без активности, после которых пользователь выходит автоматически |

- Диапазон: 5–10 080 мин (от 5 минут до 7 дней). По умолчанию: 60 мин.
- Изменение вступает в силу **немедленно**, без перезапуска сервера.
- Каждый запрос пользователя сбрасывает таймер (скользящее окно).

### JWT access-token

Отображает текущий срок жизни JWT access-token (переменная `JWT_EXPIRES_IN`). Только для чтения.

- Это **жёсткий потолок**: пользователь выйдет не позже этого срока вне зависимости от активности.
- Изменяется через `backend/.env` → `JWT_EXPIRES_IN` → перезапуск сервера (`docker compose restart backend`).
- Уже выданные токены продолжают действовать до истечения их текущего срока.

**Отличие от скользящей сессии:**

| | Скользящая сессия | JWT срок |
|-|-------------------|----------|
| Настройка | UI, без перезапуска | `.env`, требует перезапуска |
| Срабатывает | При бездействии N мин | Через фиксированный срок от выдачи |
| Можно сократить мгновенно | Да | Нет (старые токены остаются валидными) |

## Вкладка: UAT-тесты

Чеклист для приёмочного тестирования. Фильтрация по роли (показывает тесты, актуальные для вашей роли).
