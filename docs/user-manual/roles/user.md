# Роль: USER — Руководство

> Для: разработчиков, аналитиков, тестировщиков
> Обновлено: 2026-03-25

## Что может USER?

| Функция | Доступ |
|---------|--------|
| Просмотр всех задач проекта | ✓ |
| Изменение своих задач (где assignee или creator) | ✓ |
| Смена статуса любой задачи | ✓ |
| Учёт времени (таймер + ручной ввод) | ✓ |
| Комментарии к задачам | ✓ |
| Просмотр Kanban-доски | ✓ |
| Назначение задач другим | — (MANAGER, ADMIN) |
| Создание проектов / спринтов | — |
| Панель администратора | — |

## Типичный рабочий день разработчика

1. Войти в систему → Проекты → выбрать проект
2. Фильтр «Исполнитель: Я» → найти свои задачи
3. Открыть задачу → нажать **▶ Старт** → начать работу
4. По завершении → **■ Стоп** → сменить статус на «На ревью»
5. Написать комментарий о том, что сделано
6. В конце дня → **Моё время** → проверить записи

## Работа с комментариями

- Добавить: карточка задачи → вкладка Комментарии → написать → **Отправить**
- Редактировать: своей комментарий → иконка карандаша
- Удалить: свой комментарий → иконка корзины
