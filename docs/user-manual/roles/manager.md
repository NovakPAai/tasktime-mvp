# Роль: MANAGER — Руководство

> Для: PM, тимлидов, руководителей проектов
> Обновлено: 2026-03-25

## Что может MANAGER?

| Функция | Доступ |
|---------|--------|
| Создание проектов | ✓ |
| Создание и управление задачами (все задачи проекта) | ✓ |
| Назначение исполнителей | ✓ |
| Создание/запуск/закрытие спринтов | ✓ |
| Управление релизами | ✓ |
| Управление командами | ✓ |
| Просмотр отчётов и аудит-лога | ✓ |
| Bulk-операции над задачами | ✓ |
| Управление пользователями | — (только ADMIN) |
| Системный мониторинг | — |

## Типичный рабочий день PM

1. **Утро:** Открыть Дашборд → проверить активный спринт
2. **Стендап:** Доска → фильтр «Исполнитель: по одному» → обсудить статусы
3. **Планирование:** Спринты → создать спринт → назначить задачи из бэклога
4. **Назначение задач:** Открыть задачу → поле Исполнитель → выбрать
5. **Отчёт:** Администрирование → Отчёты → выбрать проект → экспортировать данные
