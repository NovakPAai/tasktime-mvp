# Роль: VIEWER — Руководство

> Для: CIO, директоров, внешних аудиторов
> Обновлено: 2026-03-25

## Что может VIEWER?

| Функция | Доступ |
|---------|--------|
| Просмотр всех проектов и задач | ✓ |
| Просмотр Kanban-доски | ✓ |
| Просмотр спринтов и релизов | ✓ |
| Просмотр отчётов (задачи по статусам, по исполнителям) | ✓ |
| Просмотр системных метрик (дашборд) | ✓ |
| Просмотр аудит-лога | ✓ |
| Создание / редактирование чего-либо | — |

## Типичный сценарий для CIO

1. Войти → Проекты → выбрать проект
2. Просмотреть Kanban-доску — общая картина по задачам
3. Администрирование → Отчёты → выбрать проект → посмотреть статистику
4. Администрирование → Активность → просмотреть последние действия в системе

VIEWER — это роль «только чтение». Никаких кнопок создания или редактирования не отображается.
